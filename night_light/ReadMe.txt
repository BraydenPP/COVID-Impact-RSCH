This folder contains data relevant to nighttime light analysis. Data was collected by the Earth Observation Group at the Colorado School on Mines (https://eogdata.mines.edu/products/vnl/) at (https://eogdata.mines.edu/nighttime_light/monthly/v10/).

GeoTIF files are very large and are omitted for up/downloading convenience.


Filing/naming system:
	2020_vcmcfg - VIIRS Cloud Mask data for year 2020.
	01-2020_tile3.avg_rade9h (omitted)- A geoTIF file displaying the average radiance recorded across a given month (January 2020) with data rounded to the houndredth decimal. "tile3" indicates the geoTIF holds data for the tile of the world map in which India falls.
	dist_light_19 and _20 - Contain values for the district-wide average of the average radiance recorded across a given month, for each district in each month of 2019 and 2020, respectively.
